    ATM Error Code Lookup

    E001
    Cash Dispenser Error - Check cash cassette and dispenser mechanism
    E002
    Card Reader Error - Clean card reader or replace if necessary
    E003
    Receipt Printer Error - Check paper supply and printer mechanism
    E004
    Communication Error - Check network connection and phone line
    E005
    Keypad Error - Clean keypad or check connections
    E006
    Power Supply Error - Check power connections and voltage
    E007
    Memory Error - System memory issue, may require restart
    E008
    Sensor Error - Check door sensors and alignment
